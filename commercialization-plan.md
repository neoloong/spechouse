# SpecHouse 商业化计划

## 📊 当前状态

- **URL**: https://spechouse.vercel.app
- **定位**: 房产界的 GSMarena × Levels.fyi
- **核心功能**: 房源搜索 + 对比 + 噪音/犯罪数据 + 投资评分

---

## 🎯 商业模式

### 模式 1: Freemium (推荐)

| 层级 | 功能 | 价格 |
|------|------|------|
| Free | 搜索 + 基本对比 + 基础分数 | $0 |
| Pro | AI 智能推荐 + 深度分析 + PDF 导出 + 投资组合追踪 | $9.99/月 |
| Agent | 房源挂牌管理 + 客户分享链接 + CRM | $29.99/月 |

### 模式 2: Lead Generation (补充)

- 免费搜索 → 导出买家信息 → 卖给房产经纪人
- 按 lead 收费 $10-50/条

---

## 📈 获客渠道

### 1. SEO + 内容 (长期)
- 城市指南: "San Francisco 买房指南 2026"
- 博客: 房产投资分析、房价趋势
- Schema markup → Google Rich Results

### 2. 社区 (短期)
- **Reddit**: r/realestate, r/Austin, r/SF
- **Twitter/X**: @spechouse, 房产博主
- **一亩三分地**: 海外华人房产讨论

### 3. 合作
- 房产经纪人 (Agent 计划)
- 房贷公司 (affiliate)
- 装修/家政服务 (导流)
